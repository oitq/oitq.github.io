配置解释 #

bot配置为一个对象，包含以下属性：

属性类型必填描述
uinnumber是登录账号
passwordstring否登录密码，不填则扫码登录(扫码登录需配合@oitq/plugin-console才能使用，且具有一定局限性)
configConfig否oicq Client配置
masternumber否bot所有者账号
adminsnumber[]否bot管理员账号

默认配置 #

{
  "admins":[],
  "config":{
    "data_dir":"./data"
  },
  "master":1659488338
}
